CSR Address Mapping Convention

ж ҮеҮҶRISCвҖ”V ISAдҪҝз”Ё12жҜ”зү№зј–з Ғ4096дёӘCSRгҖӮе…¶дёӯcsr[11:10]ж Үеҝ—CSRеҜ„еӯҳеҷЁзҡ„иҜ»еҶҷжқғйҷҗпјҢcsr[9:8]ж Үеҝ—еҸҜд»Ҙж“ҚдҪңCSRеҜ„еӯҳеҷЁзҡ„зү№жқғзӯүзә§гҖӮе…·дҪ“CSRеҜ„еӯҳеҷЁзҡ„еҠҹз”ЁйңҖиҰҒжҹҘиЎЁгҖӮ
еҜ„еӯҳеҷЁеҶ…е®№зҡ„зү№ж®Ҡеӯ—ж®өи§„е®ҡ

  1. WPRI: иҪҜд»¶еҝҪз•Ҙд»Һиҝҷдәӣеӯ—ж®өдёӯиҜ»еҸ–зҡ„еҖјпјҢеҜ№еҜ„еӯҳеҷЁеҶҷе…Ҙж—¶д№ҹеҝҪз•ҘиҜҘеӯ—ж®өпјӣ
  2. WLRL: иҪҜд»¶еҸӘжңүеҶҷе…ҘеҗҲжі•еҖјеҶ…е®№ж—¶пјҢдёӢдёҖж¬Ўзҡ„иҜ»еҸ–з»“жһңжүҚдёәеҶҷе…ҘеҖјпјҢеҗҰеҲҷиҜ»еҸ–зҡ„з»“жһңжҳҜйқһжі•еҶҷе…ҘеҖјдёҺеҶҷеүҚзҡ„еҺҹеҖје…ұеҗҢеҶіе®ҡзҡ„з»“жһңпјӣ
  3. WARL: иҪҜд»¶еҸӘжңүеҶҷе…ҘеҗҲжі•еҖјеҶ…е®№ж—¶пјҢдёӢдёҖж¬Ўзҡ„иҜ»еҸ–з»“жһңжүҚдёәеҶҷе…ҘеҖјпјҢеҗҰеҲҷиҜ»еҸ–зҡ„з»“жһңжҳҜйқһжі•еҶҷе…ҘеҖјдёҺhartзҡ„зҠ¶жҖҒе…ұеҗҢеҶіе®ҡзҡ„з»“жһңгҖӮ
    Tips:
  4. еҰӮжһңеҜ№жҹҗдёӘCSRзҡ„еҶҷе…ҘпјҢеҜјиҮҙеҸҰдёҖдёӘCSRжҹҗдёӘеӯ—ж®өзҡ„еҗҲжі•еҸ–еҖјйӣҶеҗҲеҸҳеҢ–пјҢеҲҷйңҖиҰҒи®©иҝҷдёӘCSRзҡ„жҹҗеӯ—ж®өз«ӢеҲ»еңЁе…¶ж–°еҗҲжі•еҸ–еҖјйӣҶеҗҲеҶ…иҺ·еҫ—дёҖдёӘеҖјпјҢдёҚи®әеҜ№иҜҘеӯ—ж®өиҖҢиЁҖеүҚеҸ–еҖјжҳҜеҗҰеңЁеҗҺеҸ–еҖјйӣҶеҗҲдёӯеҗҲжі•пјӣ
  5. еҰӮжһңCSRзҡ„е®ҪеәҰеҸ‘з”ҹеҸҳеҢ–пјҢзӣёе…іеӯ—ж®өзҡ„еҖјдјҡж №жҚ®зү№е®ҡз®—жі•и®Ўз®—иҺ·еҫ—гҖӮ
  6. д»»дҪ•еҜ№CSRзҡ„йҡҗејҸиҜ»еҸ–пјҢйғҪе’ҢеңЁи¶іеӨҹжқғйҷҗдёӢиҝӣиЎҢзҡ„жҳҫејҸиҜ»еҸ–еҫ—еҲ°дёҖж ·зҡ„з»“жһңпјӣ
  7. 1,2дёӯCSRеҖјзҡ„еҸҳеҢ–е№¶дёҚжҳҜдёҖз§ҚеҜ№CSRзҡ„иҜ»еҶҷпјҢдёҚдјҡеҮәи§ҰеҸ‘д»»дҪ•еүҜдҪңз”ЁгҖӮ

Machine-Level CSRs

misa

WARL read-write еҜ„еӯҳеҷЁпјҢж Үеҝ—ж ёеҝғж”ҜжҢҒзҡ„ISAз§Қзұ»пјҢMSLеӯ—ж®өиЎЁзӨәISAзҡ„е®ҪеәҰпјҢExetionsionеӯ—ж®өmisa[25:0]з”Ё0-26иЎЁзӨәA-Zеӯ—жҜҚпјҢз”ЁжқҘиЎЁзӨәж”ҜжҢҒзҡ„ж ҮеҮҶжү©еұ•з§Қзұ»гҖӮ

marchid

read-only еҜ„еӯҳеҷЁпјҢжҸҗдҫӣж ёеҝғзҡ„еҹәзЎҖеҫ®дҪ“зі»з»“жһ„еһӢеҸ·

mimpid

read-only еҜ„еӯҳеҷЁпјҢжҸҗдҫӣеӨ„зҗҶеҷЁеһӢеҸ·зҡ„е”ҜдёҖзј–з ҒпјҢиҜҘзј–з ҒеҸҚжҳ RISC-VеӨ„зҗҶеҷЁзҡ„и®ҫи®ЎпјҢиҖҢдёҚеҢ…жӢ¬еӨ–еӣҙзі»з»ҹ

mhartid

read-only еҜ„еӯҳеҷЁпјҢж Үеҝ—еҪ“еүҚиҝҗиЎҢд»Јз Ғзҡ„зЎ¬д»¶зәҝзЁӢзҡ„IDпјҢеңЁдёҖдёӘж ёеҝғеҶ…иҮіе°‘еӯҳеңЁ0еҸ·зЎ¬д»¶зәҝзЁӢпјҢеңЁиҝҗиЎҢзҺҜеўғдёӯпјҢж ёеҝғзј–з Ғе”ҜдёҖгҖӮ

mstatus & mstatush

read-write еҜ„еӯҳеҷЁпјҢз”ЁдәҺи·ҹиёӘе’ҢжҺ§еҲ¶зәҝзЁӢеҪ“еүҚзҡ„жү§иЎҢзҠ¶жҖҒгҖӮеңЁRV32дёӢпјҢйғЁеҲҶзҠ¶жҖҒдҝқеӯҳеңЁmstatushеҜ„еӯҳеҷЁдёӯгҖӮ

Global Interrupt Enable Bits

MIE е’ҢSIE еҲҶеҲ«жҳҜ Machine Modeе’ҢSupervisor ModeдёӢзҡ„е…ЁеұҖдёӯж–ӯдҪҝиғҪдҪҚгҖӮеҪ“зәҝзЁӢеңЁжқғйҷҗзӯүзә§xдёӢиҝҗиЎҢж—¶пјҢй«ҳжқғйҷҗзӯүзә§зҡ„е…ЁеұҖдёӯж–ӯжҖ»жҳҜеҗҜз”Ёзҡ„пјҢдҪҺжқғйҷҗзӯүзә§зҡ„е…ЁеұҖдёӯж–ӯжҖ»жҳҜзҰҒжӯўзҡ„пјҢж №жҚ®xIEеҗҜз”Ёе’ҢзҰҒз”ЁеҪ“еүҚжқғйҷҗзӯүзә§зҡ„е…ЁеұҖдёӯж–ӯгҖӮеҰӮжһңжғіиҰҒеңЁиҫғдҪҺжқғйҷҗиҝҗиЎҢзҠ¶жҖҒдёӢзҰҒз”Ёиҫғй«ҳзү№жқғзӯүзә§зҡ„е…ЁеұҖдёӯж–ӯпјҢйңҖиҰҒи®©иҫғй«ҳжқғйҷҗд»Јз ҒеҺ»еҚ•зӢ¬и®ҫзҪ®иҜҘдёӯж–ӯзҡ„дҪҝиғҪдҪҚгҖӮ
дёәдәҶж”ҜжҢҒдёӯж–ӯеөҢеҘ—пјҢеҲ©з”ЁxPIEдҝқеӯҳtrapеүҚдёӯж–ӯж—¶иғҪдҪҚзҡ„еҖјпјҢxPPдҝқеӯҳtrapеүҚзҡ„зү№жқғзӯүзә§гҖӮжҜ”еҰӮпјҢеҪ“д»Һзү№жқғжЁЎејҸyи·іиҪ¬еҲ°зү№жқғжЁЎејҸxж—¶пјҢжҠҠxPIEи®ҫзҪ®дёәxIEзҡ„еҖјпјҢжҠҠxIEи®ҫзҪ®дёә0пјҢжҠҠxPPи®ҫзҪ®дёәyгҖӮеҜ№еә”зҡ„пјҢеҶҚжү§иЎҢxRETжҢҮд»Өж—¶пјҢжҠҠxIEзҡ„еҖји®ҫзҪ®дёәxPIEпјҢжҠҠxIEи®ҫзҪ®дёә1,жҠҠзү№жқғжЁЎејҸеҲҮжҚўдёәyпјҢжҠҠxPPи®ҫзҪ®дёәжңҖе°Ҹзү№жқғжЁЎејҸгҖӮ

Memory Privilege

MPRV(Modify Privilege)дҪҚеҶіе®ҡloadе’Ңstoreжү§иЎҢзҡ„е…ЁзәҝгҖӮиӢҘMPRV=0,loadе’ҢstoreдҪҝз”ЁеҪ“еүҚзү№жқғжЁЎејҸдёӢзҡ„ең°еқҖзҝ»иҜ‘е’ҢдҝқжҠӨжңәеҲ¶пјӣиӢҘMRPV=1,looadе’Ңstoreзҡ„еҶ…еӯҳең°еқҖе°ҶжҢүз…§е°ҶеҪ“еүҚзү№жқғжЁЎејҸе’Ңи®ҫзҪ®дёәMPPзҡ„ж–№ејҸиў«зҝ»иҜ‘е’ҢдҝқжҠӨпјҢе№¶дҪҝз”Ёеӯ—иҠӮеәҸгҖӮжҢҮд»Өең°еқҖзҡ„иҪ¬жҚўе’ҢдҝқжҠӨдёҚеҸ—MPRVи®ҫзҪ®зҡ„еҪұе“ҚгҖӮжү§иЎҢSRETе’ҢMRETиҝҷдёӨдёӘе°ҶеҸҳжӣҙзү№жқғжЁЎејҸеҲ°дҪҺдәҺM-modeзҡ„жЁЎејҸдёҠзҡ„жҢҮд»Өж—¶пјҢдјҡжҠҠMPRVзҪ®дёә0гҖӮ

MXR(Make eXecutable Readable)дҪҚеҶіе®ҡloadжҳҜеҗҰеҸҜд»Ҙи®ҝй—®иҷҡжӢҹең°еқҖгҖӮеҪ“MXR=0ж—¶пјҢеҸӘжңүжқҘиҮӘж Үи®°дёәreadableзҡ„йЎөзҡ„loadжҢҮд»ӨжүҚиғҪжҲҗеҠҹпјӣеҪ“MXR=1ж—¶пјҢжқҘиҮӘж Үи®°дёәreadableжҲ–executableзҡ„йЎөйғҪеҸҜд»ҘжҲҗеҠҹгҖӮеҪ“жңӘеҗҜз”ЁеҹәдәҺйЎөйқўзҡ„иҷҡжӢҹеҶ…еӯҳж—¶пјҢMXRжІЎжңүд»»дҪ•ж•ҲжһңгҖӮ
