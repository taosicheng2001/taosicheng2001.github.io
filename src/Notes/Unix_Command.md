# Unix_Command

## awk
Usage

`awk [options] 'selection_criteria {action}' input-file`

Option

- `-F` Specify a file seperator

## rsync
Usage 

`rsync [option] .. [SRC] [DEST] `

`[SRC]` and `[DEST]` can be extend to `[USER@]HOST:[SRC/DEST]`

Option

- `-r` Transfer a directory 
- `-z` Compressed mode
